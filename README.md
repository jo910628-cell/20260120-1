# ✅ 스마트 수합 점검 시스템: 체크메이트 (Check-Mate)

## 1. 프로젝트 소개
- **기존 문제점:** 매일 반복되는 숙제 및 유인물 수합 확인의 번거로움.
- **해결 방안:** 모바일 터치 한 번으로 체크하고 구글 시트에 실시간 기록하는 웹앱.

## 2. 주요 기능 (필수사항 충족)
- **업무 경감:** 수기 체크 불필요, 미제출자 실시간 필터링 기능.
- **데이터 지속성:** Google Spreadsheet와 연동하여 모든 기록을 누적 저장.
- **범용성:** 시트의 학생 명단만 수정하면 어느 학급에서나 사용 가능.

## 3. 기술 스택
- Google Apps Script (GAS)
- HTML5, CSS3 (Bootstrap)
- Google Spreadsheet API

## 4. 시연 링크
- [웹앱 URL 입력]
- [데이터 확인 시트 URL 입력]
